# vue-dj-gantt

Simple Gantt diagram component for Vue.

Inspired by: https://www.npmjs.com/package/gantt-vue \
Created with: https://www.npmjs.com/package/vue-sfc-rollup

## Features

- zoom ( CTRL + scroll ),
- resize,
- drag (only in row).

## Install and use

```
npm i vue-dj-gantt
```
